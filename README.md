# Visual-Click-Guide

Turn textual software instructions into renderable SVG visual guides.

- **English Documentation**: [docs/README.md](./docs/README.md)
- **中文文档**: [docs/README.zh-CN.md](./docs/README.zh-CN.md)
- **Contributing**: [docs/CONTRIBUTING.md](./docs/CONTRIBUTING.md)
- **License**: [MIT](./LICENSE)

## Quick Start

1. Copy the prompt from [`prompt/visual-click-guide-prompt.txt`](./prompt/visual-click-guide-prompt.txt).
2. Inject it as a system prompt into Claude, ChatGPT, Doubao, or any compatible AI.
3. Ask for a visual guide in natural language.
4. Receive raw SVG code ready to render.
